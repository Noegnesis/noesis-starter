Read today's daily note at daily/YYYY-MM-DD.md (use today's actual date).
If it doesn't exist, create it with this template:

# [Today's Date]

## Top of Mind

## Today's Focus

## Linked Today

## Notes

Then:

1. Check the inbox/ folder and list any unprocessed files found.
2. Reconcile **## Linked Today** — the same-day hub. Find notes created today
   (by filename date or file creation time) anywhere in the vault that aren't
   linked there yet, and add one `[[link]]` per note, grouped by top-level
   folder (e.g. **projects:**, **research:**). Skip daily notes themselves.
   This keeps one place that answers "what did I make today?"
3. Read the most relevant active project or client folder for context.
4. Summarize the top 3 priorities for today based on recent notes.

Ask: "What are we working on today?"
